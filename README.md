# Laravel + React Boilerplate

![Project Type](https://img.shields.io/badge/type-boilerplate-blue?style=flat-square)

A modern, type-safe Laravel + React starter kit utilizing Inertia.js to bridge the gap between backend routing and frontend views. Featuring secure, headless authentication via Laravel Fortify, pre-configured ShadcnUI components, and an isolated, production-ready local Docker environment powered by Laravel Sail.
